# Bitcoin Price Predictor 

A simple LSTM based project for predicting Bitcoin (BTC-USD) closing prices, built with TensorFlow/Keras. 
## Features

- Predict next-day BTC price using a trained LSTM model
- Predict price for a future date by iteratively forecasting forward
- Update the model with fresh price data pulled from Yahoo Finance
- Uses a 200-day rolling window of closing prices as input

## Project Structure

```
.
├── model.ipynb      # Notebook: data prep, LSTM training, evaluation
├── input.py          # CLI app: predict / update using the trained model
├── btc.csv           # Historical BTC-USD price data
└── btc_model.h5       # Trained Keras model (generated by the notebook)
```

## Model Architecture

The model is a stacked LSTM network trained on 70% of historical closing price data scaled with MinMaxScaler, evaluated on the remaining 30%:

```
LSTM(50, activation='relu', return_sequences=True) → Dropout(0.2)
LSTM(60, activation='relu', return_sequences=True) → Dropout(0.3)
LSTM(80, activation='relu', return_sequences=True) → Dropout(0.4)
LSTM(120, activation='relu')                       → Dropout(0.5)
Dense(1)
```

Trained with the Adam optimizer and mean squared error loss over 50 epochs, using a 200-day lookback window.

If you don't already have `btc_model.h5`, open `model.ipynb` and run all cells. This will:
1. Download BTC-USD historical data via `yfinance`
2. Save it to `btc.csv`
3. Train the LSTM model
4. Save the trained model to `btc_model.h5`

```
Welcome to the Bitcoin Predictor!
Last update : <date>
1. Predict Bitcoin Price
2. Exit
3. Predict by Date
4. Update Model
```

- **1 — Predict Bitcoin Price**: predicts the next day's closing price using the last 200 days of data.
- **2 — Exit**: quits the program.
- **3 — Predict by Date**: enter a future date (`YYYY-MM-DD`) and the model will iteratively forecast forward, day by day, until it reaches that date.
- **4 — Update Model**: re-downloads the latest BTC-USD data from Yahoo Finance, overwrites `btc.csv`, and fine-tunes the existing model on the new data.

## Data

`btc.csv` contains daily BTC-USD OHLCV data (`Date, Close, High, Low, Open, Volume`) sourced from Yahoo Finance via `yfinance`.

## Notes / Known Limitations

- Predictions are based purely on historical closing prices (a univariate time series) — no external market signals are used.
- Multi-day forecasts (option 3) compound prediction error the further out the date is, since each step feeds the previous prediction back in as input.
- This tool is for educational/experimental purposes only and **should not be used for real financial decisions**.
